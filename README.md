# Console Chess Project

Консольные шахматы на C++17 в одном файле `main.cpp`.

## Возможности

- стандартная начальная позиция;
- ввод ходов в формате `e2 e4`;
- проверка легальности ходов;
- шах, мат и пат;
- рокировка;
- взятие на проходе;
- превращение пешки (`e7 e8 q`);
- обработка ошибок через исключения;
- тесты через CTest.

## Сборка

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Запуск

```bash
./chess
```

На Windows после сборки обычно:

```bash
chess.exe
```

## Тесты

```bash
ctest --output-on-failure
```

## Примеры ходов

```text
e2 e4
e7 e5
g1 f3
b8 c6
```

Для выхода:

```text
exit
```
